# Advanced Scientific Calculator

An advanced scientific calculator with a sleek, modern design, featuring both basic and scientific functions. The calculator provides a smooth user experience with intuitive controls, history tracking, and customizable themes.

## Key Features

### 1. Basic Operations
- Addition, Subtraction, Multiplication, Division
- Decimal input and basic arithmetic operations

### 2. Scientific Functions
- Square, Square Root
- Trigonometric functions (Sine, Cosine, Tangent)
- Exponential and Logarithmic functions
- Option to switch between Degrees and Radians for trigonometric calculations

### 3. Memory Functions
- Memory Clear (MC)
- Memory Recall (MR)
- Memory Add (M+)
- Memory Subtract (M-)

### 4. User Interface
- **Dynamic Layout:** Adapts seamlessly whether scientific features are turned on or off.
- **Responsive Design:** Fully responsive layout that adjusts to different screen sizes and devices.
- **Themes:** Light and Dark mode toggle for enhanced visual comfort.

### 5. History Tracking
- Tracks and displays a history of calculations.
- Option to clear history with a dedicated button.

### 6. Enhanced User Experience
- **Smooth Animations:** Animated button transitions for a more engaging experience.
- **Modern Font:** Utilizes sleek fonts for a futuristic look.
- **Error Handling:** Provides clear feedback for invalid operations.
- **Customizable Buttons:** Easy to read and interact with, featuring modern styles and effects.

### 7. Keyboard Support
- Supports keyboard input for digits, operators, and special keys.

## Installation

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/your-username/advanced-scientific-calculator.git
   ```

2. Navigate to the project directory:
   ```bash
   cd advanced-scientific-calculator
   ```

3. Open `index.html` in your preferred web browser to start using the calculator.

## Usage

1. **Basic Calculation:**
   - Click on numeric buttons to enter numbers.
   - Use operators (+, -, *, /) to perform basic arithmetic operations.
   - Press `=` or Enter to compute the result.

2. **Scientific Functions:**
   - Toggle scientific mode using the checkbox.
   - Select functions such as square root, sine, cosine, etc., for advanced calculations.

3. **Memory Functions:**
   - Use memory buttons to store, recall, add, or subtract values in memory.

4. **History:**
   - View calculation history and clear it using the history panel.

5. **Theme Switching:**
   - Toggle between light and dark themes for a personalized experience.
